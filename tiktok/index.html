大地堂治療院 TikTok 自動投稿システム

本ページは、大地堂治療院(東京都文京区)が運用する TikTok 自動投稿システムの公式案内ページです。当院が制作した自院のプロモーション動画を、当院が所有する公式 TikTok アカウントへ、あらかじめ定めたスケジュールで自動的に投稿するための社内向けアプリケーションです。

このシステムでできること

  • 当院が制作した動画を、当院の公式 TikTok アカウントへ自動投稿
  • 投稿前に、アカウントの公開範囲・投稿可能な動画の長さを確認
  • 対象は当院が所有する 1 アカウントのみ。第三者のアカウントやデータは扱いません

利用する TikTok 連携機能

  • Login Kit:当院アカウントの認可(OAuth 2.0)に使用
  • Content Posting API(video.publish / video.upload):当院制作動画の投稿に使用

データの取り扱い

取得したアクセストークン等は暗号化された領域に安全に保管し、第三者への提供・販売は一切行いません。詳細は下記をご覧ください。

About (English)

This is the official information page for the TikTok auto-posting system operated by Daichido Clinic (Bunkyo-ku, Tokyo, Japan). It automatically publishes the clinic's own promotional videos to the clinic's own official TikTok account on a predefined schedule. It uses TikTok Login Kit (OAuth 2.0) to authorize the clinic's own account, and the Content Posting API (video.publish / video.upload) to publish the clinic's own videos. It operates on a single account owned by the clinic and does not access or process any third-party user data. Access tokens are stored securely and are never sold or shared.
